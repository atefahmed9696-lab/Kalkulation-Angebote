# Archiv – Versionshistorie

Dieses Verzeichnis enthält automatisch gespeicherte Sicherungskopien der Hauptdateien.

## Namensschema

```
index.html_YYYY-MM-DD_HH-MM.bak
app.html_YYYY-MM-DD_HH-MM.bak
```

Das Datum/Uhrzeit-Präfix gibt an, **wann die Sicherung angelegt wurde** (also den Stand VOR der jeweiligen Änderung).

## Wiederherstellung

Um eine alte Version wiederherzustellen, einfach die gewünschte `.bak`-Datei in das Hauptverzeichnis kopieren und umbenennen:

```bash
cp archiv/index.html_2026-04-24_22-59.bak index.html
```
